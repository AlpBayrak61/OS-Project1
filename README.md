To compile:

```bash
make
```

To clean:

```bash
make clean
```

To run:

```bash
./mysumfix <n> <m> <input_file> <output_file>
```

Example:

```bash
./mysumfix 114 4 input.txt output.txt
```

The current input file (`input.txt`) contains 114 numbers.
